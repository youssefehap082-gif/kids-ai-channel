# Kids AI Channel — Auto Uploader
Daily automation: 3 long videos (≥3 min) + 6 Shorts about animals/birds/reptiles/fish (English).
- 10 facts per long video (OpenAI)
- Voiceover (OpenAI TTS), alternating voices
- B-roll from Pexels/Pixabay
- SEO (titles/descriptions/tags/hashtags)
- Scheduled to US peak times
- No repetition of animals

Secrets needed:
OPENAI_API_KEY, PEXELS_API_KEY, PIXABAY_API_KEY, YT_CHANNEL_ID, YT_CLIENT_ID,
YT_CLIENT_SECRET, YT_REFRESH_TOKEN
(Optional: HF_API_TOKEN, REPLICATE_API_TOKEN)

Music: put royalty-free MP3s into assets/music/ and set USE_BG_MUSIC=true (optional).
